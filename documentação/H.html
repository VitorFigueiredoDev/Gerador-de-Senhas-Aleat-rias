Descrição dos Elementos

        <DOCTYPE>: Define a versão do HTML utilizada no documento.
        Elemento HTML: Define o início de um documento HTML.
        Cabeçalho   (head): Contém informações sobre o documento, como o título e metadados.
        Meta Charset: Especifica o conjunto de caracteres utilizado (UTF-8).
        Meta Viewport: Define como a página deve ser renderizada em dispositivos móveis.
        Título: Define o título da página exibido na aba do navegador.
        Link CSS: Importa o arquivo CSS externo.
        Corpo  (body): Contém o conteúdo visível da página.
        Cabeçalho  (header): Seção de cabeçalho da página.
        Navegação  (nav): Contém links de navegação para a documentação HTML, CSS e JavaScript.
        Separadores  (div.separator): Divisões visuais no cabeçalho.
        Lista de Links  (ul): Lista não ordenada de links.
        Item de Lista  (li): Cada item da lista de links.
        Âncoras  (a): Links para os documentos de documentação.
        Conteúdo Principal  (div.container): Contém o conteúdo principal da página.
        Título Principal  (h1): Título principal da aplicação.
        Configurações  (div.settings): Seção onde o usuário pode configurar a geração de senhas.
        Campos de Entrada  (input): Entradas para o comprimento da senha e seleção de tipos de caracteres.
        Botão de Geração  (button#generate): Botão para gerar uma senha aleatória.
        Resultado da Senha  (div.result): Seção onde a senha gerada é exibida.
        Força da Senha  (div#passwordStrength): Indicador visual da força da senha.
        Botão de Cópia  (button#copyPassword): Botão para copiar a senha gerada para a área de transferência.
        Script JavaScript  (script.js): Arquivo JavaScript para interatividade e funcionalidade da aplicação.
